# Instruksi Instalasi

1. Install XAMPP
2. Install PHP 8.1
3. Install Composer
4. Buka XAMPP, Jalankan apache dan phpmyadmin
5. Buka PHPMyAdmin di browser, buat database baru dgn nama "dbpos"
6. Buka proyek di code editor
7. Jalankan perintah "composer install" atau "composer update" di terminal/cmd.
8. Kemudian jalankan perintah "php artisan key:generate" di terminal/cmd.
9. Kemudian jalankan perintah "php artisan migrate" di terminal/cmd.
10. Kemudian jalankan perintah "php artisan db:seed" di terminal/cmd.
11. Terakhir jalankan perintah "php artisan serve" di terminal/cmd untuk menjalankan sistem.
